# Programming-aware stopword supplement for vectr symbol-boost guard.
# Purpose: covers short common-programming words that the length guard (min_leaf_len=4)
# would otherwise let through because they are exactly 2-3 chars but appear casually
# in natural-language queries (e.g. "get", "set", "run", "add").
# Also covers programming abbreviations/identifiers that are <= 3 chars and would
# match accidentally (idx, len, str, int, id, db, ok, etc.).
# One word per line. Comments (#) and blank lines are stripped by the loader.
# Do NOT add words longer than 3 chars here (3-char section) — the length guard handles those;
# this list exists specifically for the short words that slip through.
#
# IMPORTANT: this file feeds ONLY symbol_identity_boost (chunk_quality.py) and the
# forced-inclusion token guard (searcher.py) — NOT BM25 tokenisation or embedding.
#
